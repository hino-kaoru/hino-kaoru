“砂漠が美しいのは、どこかに井戸を隠しているからだよ”

伝えたいこと、大切にしたいこと、
想いがたくさんあふれて、考えがまとまらない状態はまるでゴールが見えない砂漠の中を
さまよっているようです。
広い砂漠の中からオアシスを見つける旅のように、
本当に伝えたい大切なことを一緒に探して、
伝えたい相手へ届けられるよう
お手伝いをさせて頂きたいです。
